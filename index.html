核心功能说明
了解希沃多媒体在课堂中的主要能力与使用场景
电子白板功能
板书与演示
提供类似黑板的书写体验，并支持多页板书、图形绘制与内容保存，适合数学推导、语文批注等场景。
  • 支持手写笔与手指触控书写
  • 多页白板，可前后翻页
  • 图形工具：直线、矩形、圆形等
  • 一键保存板书为图片或课件
多媒体播放
音视频与课件
集成常用多媒体播放能力，支持视频、音频、图片、PPT 等教学资源的快速打开与展示。
  • 支持本地与U盘文件播放
  • 常见格式：MP4、MP3、JPG、PPT 等
  • 播放过程中可随时暂停与批注
  • 适合微课、实验演示、纪录片片段等
屏幕批注
任意界面标注
无论是网页、PPT 还是其他软件界面，都可以一键进入批注模式，进行圈画、标记与说明。
  • 一键开启/退出批注模式
  • 支持多种颜色与笔迹粗细
  • 可截图保存当前批注内容
  • 适合讲解试题、分析图表等
课堂互动工具
提升参与度
内置计时器、随机点名、抢答器等工具，帮助老师组织课堂活动、控制节奏、提高学生参与度。
  • 计时器：小组讨论、限时作答
  • 随机点名：公平抽取学生回答
  • 抢答器：适合竞赛与课堂游戏
  • 可与班级管理功能配合使用
课件与资源管理
教学资源
支持课件导入、分类与简单管理，部分版本可与云端账号同步，实现多教室共享资源。
  • 支持从U盘或本地导入课件
  • 按学科或章节进行简单分类
  • 部分版本支持云端账号登录
  • 方便不同教室之间共享资源
账号与权限
安全与管理
通过教师账号登录，可实现个性化设置与资源同步；同时保障设备使用安全与数据隐私。
  • 教师可使用个人账号登录
  • 部分设置仅管理员可更改
  • 建议下课后退出账号
  • 重要数据建议定期备份
使用流程与操作指南
从上课前准备到下课收尾的完整步骤
上课前 · 设备检查
1. 打开多媒体电源与主机电源。
2. 确认希沃多媒体系统已启动。
3. 简单测试触控书写是否正常。
4. 播放一段音频测试音响。
上课中 · 功能使用
1. 使用电子白板进行板书与讲解。
2. 需要时打开多媒体课件或视频。
3. 使用屏幕批注强调重点内容。
4. 适时使用计时器、点名等互动工具。
下课后 · 收尾与保护
1. 保存重要板书或课件内容。
2. 退出个人账号（如有登录）。
3. 关闭多媒体系统与电源。
4. 确认教室环境整洁，设备完好。
常见问题与维护建议
遇到问题时可以先对照本节进行自查
Q1：设备无法开机或无信号怎么办？
请先检查电源插座与总电源是否打开，再确认主机电源键是否按下。如仍无反应，请联系电教组或信息中心老师处理。
Q2：触控不灵敏或偏移严重怎么办？
可尝试重启系统或进行触控校准（如系统提供该功能）。若问题持续存在，请记录教室编号并报修。
Q3：没有声音或声音很小怎么办？
检查系统音量、播放器音量以及物理音响音量旋钮是否打开；确认未插入耳机或其他音频设备。
Q4：软件打不开或运行异常怎么办？
可先重启希沃多媒体软件或重启电脑。如多次尝试仍失败，请截图或拍照错误界面，并联系技术老师。
